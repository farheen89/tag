 1- Please upload all the files and directories in any directory on your server.

 2- Edit "includes/config.inc.php" for database settings and use "includes/tbl_products.sql" for database generation
 
 3- Open browser and enter

Server Requirements
========================


 Any Linux/Windows or Other server that supports folowing:

 php, curl with ssl support, safemode "off", openbasedirectory "disabled"

 For MSN Messenger Script sockets support is required
